My Projects

Degree Dialog - AI College Advisor Chatbot

A student-facing AI chatbot and information portal for colleges built as a full-stack monorepo. The application provides real-time Q&A assistance powered by Google's Gemini AI, helping students discover college information through a clean, intuitive web interface. Frontend deployed on Vercel, backend on Render with CORS-enabled API integration.

  • React + Vite frontend with real-time chatbox UI
  • Django backend with Gemini AI API integration
  • CORS-enabled REST API for frontend-backend communication
  • College advisor AI persona with specialized prompt engineering
  • Deployed frontend on Vercel, backend on Render

Sports Analysis - Football Analytics Dashboard

A comprehensive web application for sports data analytics that visualizes football statistics across multiple leagues and teams. Built with a React + Vite frontend and Flask backend, it integrates with the Football-Data API to provide real-time league standings, team statistics, and player position analysis. Features interactive dashboards with Chart.js visualizations and Tailwind CSS styling.

  • React + Vite frontend with Tailwind CSS responsive design
  • Flask Python backend with caching to manage API rate limits
  • Live league standings for PL, La Liga, Bundesliga, Serie A, and more
  • Interactive team statistics with Bar and Pie charts using Chart.js
  • Player position analysis and competition breakdown

Mask Detection - Real-time Object Detection System

An advanced computer vision application using YOLOv8 for real-time mask detection with multi-object tracking. The system monitors compliance rates, generates audit logs, and provides visual feedback through OpenCV overlays. Trained on 4,547 images with data augmentation, it accurately tracks individuals across frames and logs detection events with confidence scores for workplace safety monitoring.

  • YOLOv8 object detection with ByteTrack multi-object tracking
  • Real-time compliance monitoring with visual progress bar
  • CSV logging of all detection events with timestamps and confidence scores
  • Audio alerts for no-mask detection with debounced notifications
  • Roboflow-trained dataset with 4,547 images and augmentation
